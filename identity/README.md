# Corporate Identity

Logo, Briefpapier und Brand Guidelines der BLAUWEISS-EDV LLC.

## Brand Colors

| Farbe | Hex | Verwendung |
|-------|-----|------------|
| **Blau (Cyan)** | `#00A5DB` | "blau" im Logo, Überschriften, Links |
| **Grün** | `#8DC63F` | "weiss" im Logo, Akzente |
| **Grau** | `#666666` | Fließtext, Footer |

## Logo

Das Logo besteht aus dem Schriftzug "blau●weiss" mit:
- "blau" in Cyan (#00A5DB)
- Grüne Punkte als Trenner
- "weiss" in Grün (#8DC63F)

### Dateien

| Datei | Format | Verwendung |
|-------|--------|------------|
| `logo/blauweiss-logo.svg` | SVG | Web, Print (skalierbar) |
| `logo/blauweiss-logo.png` | PNG | Dokumente, E-Mail-Signatur |
| `logo/blauweiss-logo-white.png` | PNG | Dunkler Hintergrund |
